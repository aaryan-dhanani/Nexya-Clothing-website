# 👕 Nexya Clothing Website

Nexya Clothing Store is a modern e-commerce web application designed to offer stylish and trendy clothing for men and women. The project demonstrates how a fully functional online store can be developed using frontend technologies along with a cloud-based database solution.

The website provides a smooth and responsive shopping experience, allowing users to browse products, add items to the cart, and interact with a clean and user-friendly interface.

Unlike traditional applications, this project uses Google Sheets as a backend database, enabling real-time data storage and retrieval without requiring a complex server setup.

---

## 📊 Project Objective

The objective of this project is to demonstrate how cloud-based tools like Google Sheets can be used as a lightweight backend for small-scale web applications. It highlights efficient data handling, real-time updates, and modern UI/UX design without the need for complex backend infrastructure.
